# Theme Image Assets

This directory contains showcase images for each design theme. These images will be displayed in the hero cards to provide visual examples of each aesthetic.

## Required Images

Place the following images in this directory:

1. **glassmorphism.jpg** - Example of frosted glass UI elements
2. **neumorphism.jpg** - Soft, extruded interface elements
3. **brutalism.jpg** - Bold, stark design with harsh contrasts
4. **bento-grid.jpg** - Grid layout with varying card sizes
5. **claymorphism.jpg** - Playful, clay-like UI elements
6. **gradient-mesh.jpg** - Smooth, flowing color transitions
7. **minimalism.jpg** - Clean, minimal design examples
8. **retro-futurism.jpg** - Neon, 80s-inspired sci-fi aesthetics
9. **dark-mode.jpg** - Sophisticated dark interface examples

## Image Specifications

- **Format**: JPG (JPEG)
- **Recommended Size**: 1200x800px (3:2 aspect ratio)
- **Quality**: High quality, 85-90% compression
- **File Size**: Optimize for web (aim for < 300KB per image)

## Fallback Behavior

If images are not provided, the application will:
- Hide the image container gracefully
- Still display the theme information
- Maintain the hero card styling
- Show only the icon and title

The application will continue to function perfectly without images, though visual examples enhance the user experience.
